# Asteroids Arcade Game

## Overview
An interactive arcade-style game built with Python and Pygame. The game features dynamic gameplay, collision detection, scoring, and power-up upgrades.

## Features
- Object-oriented game architecture
- Collision detection system
- Score tracking
- Upgrade/power-up mechanics
- Progressive difficulty

## Technologies Used
- Python
- Pygame

## How to Run
Linux/macOS:
source venv/bin/activate

Windows:
venv\Scripts\activate

pip install pygame
pip install pygame_gui

python game.py
